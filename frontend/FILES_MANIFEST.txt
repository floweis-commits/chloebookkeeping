================================================================================
TROUBLESHOOTING
================================================================================

API not found:
  → Check NEXT_PUBLIC_API_URL in .env.local
  → Ensure FastAPI backend running on port 8000

Login fails:
  → Check browser console for error messages
  → Verify credentials with backend
  → Check API response status in Network tab

Files not loading:
  → Verify API endpoint /api/files exists
  → Check user has access to tenant
  → Check token is valid (not expired)

TypeScript errors:
  → Run npm install to update packages
  → Delete .next/ and rebuild
  → Restart dev server

Types missing:
  → All types in lib/types.ts
  → Check import statements
  → Ensure types exported correctly
